# Delivery Route Optimization Project

This project solves a real-world delivery routing problem using multiple algorithms:

- Backtracking (route exploration)
- Greedy (parcel selection)
- Dynamic Programming (time validation)
- Dijkstra & MST (graph optimization)
- TSP (optimal route)

## Output
- Optimal delivery route
- Selected parcels
- Performance graphs

## Tools Used
Python, matplotlib, networkx

## Conclusion
Combining multiple algorithmic strategies provides efficient solutions to complex real-world problems.
